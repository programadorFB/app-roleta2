Arquitetura

Full-stack SPA com React 19 + Vite no frontend, Express 5 no backend, PostgreSQL para persistencia, Redis para cache e Socket.IO para real-time.

Motor Score Engine

Sistema passivo que roda no backend apos cada ciclo de fetch. Registra sinais e verifica acertos automaticamente para TODAS as mesas.

Funcionamento
  1. Apos cada fetch, processSource() analisa os novos spins
  2. Se masterScoring detecta convergencia 3+, registra sinal pendente
  3. Nos proximos 3 spins, verifica se algum numero sugerido acertou
  4. Atualiza contadores win/loss em motor_scores por modo vizinho
Modos de Vizinho
ModoCoberturaDescricao
0ExatoApenas numeros sugeridos
1+1 vizinhoSugeridos + 1 vizinho de cada lado
2+2 vizinhosSugeridos + 2 vizinhos de cada lado
